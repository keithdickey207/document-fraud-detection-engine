# Document Fraud Detection Engine v2

**Enterprise-grade, fully local-first document forensics system**

Detects:
- Metadata & structural tampering
- Layout / physical anomalies
- Radial-frequency halftone (re-photographed prints)
- Error Level Analysis (digital edits)
- Copy-Move forgery (cloned signatures, stamps, fields)

## Quick Start

```bash
pip install opencv-python-headless numpy pillow pypdf pdf2image scipy matplotlib
python document_fraud_detection_engine_v2.py your_document.pdf --output report/
```
